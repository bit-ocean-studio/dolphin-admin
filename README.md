# News App

## 技术栈

- [Vue 3](https://cn.vuejs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://cn.vitejs.dev/)
- [Naive UI](https://www.naiveui.com/)
- [Sass](https://sass-lang.com/)

## 使用

### 环境

- Node.js >=16.14.0
- pnpm

### 配置环境变量

配置 `.env` 文件。

### 安装

```bash
pnpm i
```

### 启动

```bash
pnpm dev
```

### 构建

```bash
pnpm build
```
